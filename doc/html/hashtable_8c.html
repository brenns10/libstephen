smb_ht_bckt* ht_find_in_bucket ( smb_ht_bckt *  pBucket,
DATA  key,
DATA_COMPARE  equal 
)

Finds a value within a singly linked hash table bucket list.

This function actually performs a lot of tasks rolled into one. First, if you pass NULL as the pBucket, it will return NULL. More interestingly, it will return the pointer to the bucket that equals key if it exists, or the last bucket in the list if there is none equalling key. This way, you can just check the return value, and if key == pBucket->key, you have the pre-xisting spot for key. Otherwise, you can just create the next item in the list.

Parameters:
pBucketThe bucket to search within.
keyThe key to search for.
Returns:
The bucket containing key.

void ht_insert_bucket ( smb_ht *  table,
smb_ht_bckt *  pBucket 
)

Insert a bucket with data into the hash table. If the key already exists in the table, overwrites with a new value.

This function is at a lower lever than ht_insert. In order to properly implement a hash table, the table must expand as you add to it, which is the part that ht_insert takes care of.

Parameters:
tableThe table to insert into.
pBucketThe bucket containing the data to insert. Note that this bucket may not be the final bucket containing the data.
