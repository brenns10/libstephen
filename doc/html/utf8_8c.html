wint_t utf8iter ( const char *  _src,
int *  _s 
)

A function supporting iteration through a UTF-8 string.

Use this function to iterate through a UTF-8 string on a per-codepoint basis, rather than on a per-byte basis. This is useful only if you're going to do something with the code points. Otherwise, it is wise to act in an encoding-agnostic way (as UTF-8 allows) and act in a bytewise manner.

This function uses a pointer to an iteration variable. Initially, the variable is set to the index of the next code point in the string. The funtion reads this character from the string, updates the iteration variable to hold the index of the next code point, and returns the character.

Parameters
_srcThe UTF-8 string to iterate through.
_sA pointer to the iteration variable.
Returns
The character at _src[*_s], with *_s updated.
Return values
WEOFon error
