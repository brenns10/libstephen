int read_escape ( const wchar_t *  source,
int  len,
wchar_t *  out 
)

Place the escaped wchar in source into out.

Source should be a string "\..." containing an escape sequence. It should include the backslash. This function will read the escape sequence, convert it to a wchar_t, store that in out, and return the number of characters read.

Parameters:
sourceThe string escape sequence to translate.
lenThe length of the string to read. (assumed >= 1)
outWhere to store the output character.
Returns:
The number of characters read.
Exceptions:
Sets*out to WEOF if it would readpast len.
