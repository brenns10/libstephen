void fsm_dot ( fsm *  f,
FILE *  dest 
)

Print an FSM to graphviz dot format.

Graphviz is a neat CLI tool to create diagrams of graphs. The dot format is an input language for graphs. This function takes a FSM and prints it to a file as a dot file, so that it can then be converted to a image by graphviz.

Parameters:
fThe machine to print.
destThe destination file.

fsm* fsm_read ( const wchar_t *  source,
smb_status *  status 
)

Read a Finite State Machine from its text representation.

All fsm's can be printed using fsm_print(). This function *should* be able to take any such string representation and convert it back into an equivalent fsm again.

Parameters:
sourceThe string to read in.
statusStatus variable for error reporting.
Returns:
New FSM, or NULL on error.
Exceptions:
CKY_TOO_FEW_LINESif there isn't at least one line.
CKY_MALFORMED_TRANSif there is a problem reading a transition.
